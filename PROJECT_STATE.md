# AI高考人生军师 - 当前进度

更新时间：2026-06-09

## 当前状态

- 项目已接入 Next.js + TypeScript + TailwindCSS + shadcn/ui 风格组件。
- 当前本地服务地址：http://127.0.0.1:3000
- 当前 AI 提供商已切到 DeepSeek。
- `.env.local` 已配置 DeepSeek，但 API Key 已在聊天中暴露，正式使用前建议去 DeepSeek 平台重置一个新 Key。

## 已完成

- 首页：保留短入口逻辑，无需注册，学生/家长双入口。
- 基础信息页：恢复移动端卡片式版面。
- 学生版/家长版：已分开访谈题库。
- 考生类型：普通类、艺考类、体考类。
- 艺考/体考：已增加专业分输入。
- 报告生成：已从本地规则引擎切换为 DeepSeek API。
- DeepSeek 返回结构：已做归一化兼容，避免模型字段不稳定导致前端空白。

## 当前重点问题

- 报告质量还需要继续调 prompt，让它更像“高考人生导师”，少套话，多具体学校、专业、城市、路径。
- 学校推荐仍缺真实权威数据源。后续需要接入或导入：
  - 各省一分一段表
  - 近三年投档线/专业组录取线
  - 艺考/体考综合分规则
  - 学校就业质量报告
  - 保研/升学/转专业政策
- 视觉版式以后不要大改，只在原有风格上补功能。
